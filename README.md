# Yoga Showcase Website

Welcome to the **Yoga Showcase Website**, a modern and responsive platform designed to showcase yoga courses, connect with client easily through an integrated contact system, and manage appointments efficiently.

---

## Technologies Used

- **Next.js 13** – React framework for fast and optimized web applications  
- **React** – Component-based architecture  
- **Tailwind CSS** – Utility-first CSS for rapid UI development  
- **Next/Image** – Optimized image handling for faster loading  
- **Custom Components** – Reusable buttons, containers, and navbar

---

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/Meriemchm/chm_yoga_website.git
```

2. Install dependencies:

```bash
npm install
```

2. run dependencies:

```bash
npm run dev
```

3. Run the development server:

```bash
npm run dev
```

## Folder Structure

- `components/` – Reusable components (Navbar, Buttons, etc.)  
- `app/` – Website pages (Home, Services, Contact, etc.)   
- `public/` – Static files such as images, icons, and assets
- `schemas/` – Data schemas or models used in the project
